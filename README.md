# 日光之下：加利利的一年 / Under the Sun: A Year in Galilee

基于拉纳神学的历史生活模拟RPG，背景设定在公元30年的加利利。

**在线试玩**: [https://dengxiaocheng.github.io/TheologyGame-UnderTheSun/](https://dengxiaocheng.github.io/TheologyGame-UnderTheSun/)

## 技术栈
- TypeScript + CLI
- Node.js

## 开发
```bash
npm install
npm start
```

`npm start` 通过 `tsx src/main.ts` 启动本地 CLI，无需预构建。

## GitHub Pages

公开页面部署在：

https://dengxiaocheng.github.io/TheologyGame-UnderTheSun/

Pages 入口是仓库根目录的 `index.html`。浏览器页只提供项目介绍和本地运行入口；完整游戏仍以 TypeScript CLI 运行，未被改写为浏览器版本。
